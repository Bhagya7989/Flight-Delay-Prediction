# Flight Delay Prediction using Machine Learning

## Overview
This project predicts whether a flight will be delayed using historical airline data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Random Forest
- Google Colab

## Dataset
50,000 flight records from a real-world flight dataset.

## Features Used
- Month
- Day
- Day of Week
- Airline
- Origin Airport
- Destination Airport
- Distance
- Departure Delay

## Model
Random Forest Classifier

## Accuracy
88.37%

## Results
Successfully predicted flight delays with 88.37% accuracy.# Flight-Delay-Prediction
